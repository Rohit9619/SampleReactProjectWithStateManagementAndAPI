Step 1 - Download and Install Node js from ( https://nodejs.org/en/download/ ) download LTS version.

Step 2 - Download and Install Vs Code Editor.

Step 3 - Create Shopping folder, copy shopping.rar inside shopping folder and right click to use extract here option. (ps - click exactly extract here to extract only files).

Step 4 - Open Vs code. On the top left click on files -> open folder. Select the shopping folder you created.

Step 5 - In vs code on the top go to view -> terminal (to open terminal).

step 6 - By default powershell is open in terminal. Type npm install (to install node pacakges).

step 7 - After installation Type npm start (to run the project).
